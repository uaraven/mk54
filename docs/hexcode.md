8-segment display of MK calculator did not display proper characters A to F,
instead it used weird symbols.

`mk54` project uses following hex digits to represent MK-series weidgits:

| MK symbol | hex digit |
|:---------:|:---------:|
|     -     |     A     |
|     ⎣     |     B     |
|     ⊏     |     C     |
|     Г     |     D     |
|     E     |     E     |
|           |     F     |
